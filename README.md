
## Tech Stack

- **Frontend**: Nuxt.js 3 (Vue.js, TypeScript)
- **Backend**: Supabase 
- **Deployment**: GitHub Pages (Static Site Generation)

## Getting Started

### Development
```bash
npm install
npm run dev
```

### Build & Deploy
```bash
npm run generate
npm run deploy
```
