## Frontend Development (Vite + Vue)

This project now includes a frontend application built with Vite and Vue.

### Running the Frontend Development Server

To run the Vite development server for the frontend, use the following command:

```bash
npm run dev:vite
```

This will start a local development server, typically at `http://localhost:5173`.

### Building the Frontend

To build the frontend for production, use the following command:

```bash
npm run build:vite
```

The build output will be placed in the `docs/vite` directory. This directory is included in the `gh-pages` deployment, making the frontend part of the live GitHub Pages site.
